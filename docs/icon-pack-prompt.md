# English Land Icon Pack Prompt

Usa este prompt para generar un paquete consistente de iconos PNG para Word Bingo Quest y Simon Says 6.

## Prompt

Create a complete educational icon pack for a third-grade English learning game called "English Land: Adventure Quest". Generate one separate transparent PNG icon for each vocabulary item and classroom command listed below.

Style requirements:
- Cute child-friendly 3D sticker style.
- Consistent lighting, soft shadows, rounded shapes, saturated but balanced colors.
- White or transparent background only.
- No text, no letters, no numbers written inside the icon unless the item is specifically a number or weekday.
- No watermark.
- Centered object, full object visible, not cropped.
- Square canvas, 1024 x 1024 px.
- Export as transparent PNG.
- Use the same camera angle and visual language across the full set.
- Icons must be readable at 42 px and 58 px.
- Avoid realistic photos; use polished educational game assets.

File naming:
- Use lowercase kebab-case.
- Save files in `assets/resources/generated-icons/`.
- Example: `mother.png`, `touch-your-head.png`, `there-is.png`.

## Negative Prompt

Do not include blurry icons, inconsistent art styles, cropped objects, dark backgrounds, text-heavy icons, watermarks, photorealistic images, scary faces, distorted hands, unreadable tiny details, or duplicated compositions.
